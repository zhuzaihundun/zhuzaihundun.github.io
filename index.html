整理和总结学术文献

  1. 轻整理，重搜索。对文献资料进行简单的分类；通过各种工具快速定位到所需要的文献。

  2. 做好标记。可以将一批文献放在一起写一段综述，然后加上跳转定位功能。可以通过表格的方式记录编号、题目、期刊、作者、时间、关键词、文献综述、论文方法、论文信息、分类、问题、句子 。

  4. 最好以一个项目或者文章对文章进行分类。

  5. 重点是阅读论文的输出，不用太多注重文献的分类。

  6. 这些方法也可以推广到工作领域。

睡眠革命

发表于 2019-12-19 | 分类于 杂文


  《睡眠革命》，作者尼克·利特尔黑尔斯，他是一位睡眠教练，主要是研究睡眠对运动员的发挥的影响。这本书就是他对睡眠的研究成果。
  本文的成果：
  一、睡眠修复。
  1.昼夜规律。走到户外，让太阳光调控你的睡眠习惯；了解自己的生理特征，根据自己的生活情况，调整自己的生活习惯；睡眠最好的时间是在凌晨2~3点，天亮之后休息会影响你整天的生活；早上醒来尽量放慢自己的生活习惯，太急躁的早晨习惯会拖垮自己的身体；在晚上，尽量避免蓝光，尽量使用红光或者黄光，甚至是蜡烛；让自己的生活习惯符合古代的日出而作日落而息的生活节奏。
  2.睡眠类型。了解自己的睡眠类型，这很重要，可以通过慕尼黑大学的问卷调查了解自己的类型；规划自己一天的行程，在精力最好的时候做最重要的事情；咖啡应当做高效的表现的增强剂，不要当成习惯，一天的摄入量不应超过400毫克；不要在周末睡懒觉；在办公区配置日光灯，提高工作效率，改善心态；知道何时上前，何时退后；和自己的伴侣协调好睡眠习惯。
  3.90分钟睡眠法。固定自己的起床时间，用R90确定自己的睡觉时间；不要过分注重某天晚上的睡眠时间，应该一周分析自己睡了多少个R90周期；尽量避免连续三晚睡眠不足。
  4.热身和舒缓。睡眠前后的例行程序将会影响睡眠质量和整天的工作效率；快速洗个温水浴，然后选择较凉爽的环境，有助于睡眠；一个自然唤醒灯是很有必要的，睡前从亮到黑暗，醒来从黑暗到明亮。
  5.日间休息。午后（1~3），傍晚（5~7）可以有30分钟的休息，小憩很重要；每90分钟，可以稍微让自己休息会。
  6.改造你的床铺。学会侧卧睡眠；了解什么样的床垫软硬适合你；床垫可以便宜多换，而不是一次花费很多；使用舒适透气的床上用品；床的尺寸要尽量大；床垫比床架重要。
  7.修复室。卧室是身心的修复室，只应该有与睡觉有关的东西；卧室颜色应该中性，保持房间清洁；卧室应该比其他房间凉爽
  二、R90在行动。
  8.把握时机。利用R90方法修复自己的睡眠，同时注重饮水和锻炼。
  9.与敌同眠。各种睡眠问题，比如，夜班，飞机，冬夏的季节影响。
  10.主队。伴侣和家庭对睡眠的影响。

  个人总结：1睡眠本身。利用R90睡眠法固定自己的起床时间；每天尽量争取一个小憩的时间段；每工作90分钟，休息一会；睡前醒后的例行程序。
  2.与睡眠相关。尽量减少卧室其他和无关的东西；一个模拟自然光的呼吸灯是有必要的；床要尽量大，床垫不宜太硬也不宜太软，床垫要勤换；卧室颜色中性，不应该太刺激；晚上应该以尽量减少蓝光；有一个凉爽的睡眠环境。

梯度下降理解

  梯度方向：在$x^{2}$出点的导数，随着$x$的增加，$\Delta{y}$的变化情况，根据$\Delta{y}$的情况，得到相应的$\Delta{x}$的变化情况，从而不断缩小$\Delta{y}$的值，求得最优。现有的不同的优化算法是因为对$\Delta{x}$的求解方式不同。

最小二乘法理解

  目前的机器学习很多算法都是基于最小二乘法，而最小二乘法的数学正确性是这些算法的基础。
  在机器学习中，我们得到预测结果$y^{‘}$，而真是结果为$y$，那么$y-y^{‘}$代表的是真实和预测之间的差异。只要累计所有样本之间的差异之和，使得差异之和最小，可以自然的认为这个预测算法是最好的。但这里有一些问题值的思考。
  1.$y-y^{‘}$可正可负，因此为了保证累计所有的偏差，可以是绝对值或者平方两种方法，但绝对值函数在求导的过程中有许多讨论的问题，因此使用$(y-y^{‘})^{2}$更加适合后续的求最优。因此很多函数的目标函数都是基于平方和，也就是最小二乘。
  2.这只是直观的理解。那么能不能通过严格的数学证明其想法的正确性呢？这个就是高斯的一个成就。设$\epsilon=y-y^{‘}$，其中$\epsilon$可以认为是随机分布，并且是独立同分布，我们假设分布的概率密度函数为$p(\epsilon)$，假设一个联合概率：$L(x)=p(\epsilon_{1})\dots p(\epsilon_{n})$。但 $\frac{d}{dx} L(x)=0$ 成立时，$L(x)$ 取得最大值。如果前面最小二乘法成立的话，x取得样本数的平均值，即：$\frac{d}{dx} L(x)|_{x=\overline{x}}=0$。最终求得 $p(\epsilon)=\frac{1}{\sigma \sqrt{2\pi}}e^{-\frac{\epsilon^{2}}{2\sigma^{2}}}$。而这就是正态分布，也就是说如果误差的分布是正态分布，则最小二乘法的结果就可能是最终的结果。而一般误差都是随机的、独立的、无数的，而这刚好可以运用到中心极限定理，可以证明这些误差分布符合正态分布，从而又说明最小二乘法的正确性。

智慧的提问--感触

发表于 2019-11-13 | 分类于 杂文
  讲述的是在向黑客提问时的一些礼节，但对其他领域向人请教也有一定的借鉴意义。文章链接，[点这里](https://github.com/ruby-china/How-To-Ask-Questions-The-Smart-Way/blob/master/README-zh_CN.md)。   智慧的提问：1.在提问之前应该自己查阅各种资料，自己在网上多方查找无果的情况下，可以向他人请教。   2.查找合适的提问场所，论坛、邮箱等。   3.然后向提问对象提问，应注意：尽量简洁的描述自己的问题，不要用过多的修饰词；使用“请”或者“谢谢您的回复”等之类的词语；附上自己解决问题时，所做的工作和努力；请求对方指出解决问题的方向而不应是答案。   4.提问的技巧。尽可能详细简洁的描述自己所遇到问题的场景，让被提问者迅速锁定问题场景，尽量启发式的提问，尽量避免单纯的索取知识。   自己的感悟：如何巧妙的提问？让被提问者感兴趣，这样才能获得更加优质的回答。并且，提问应该偏向引导性的回答，尽量不要二元答案，除非必要。

机器学习的评价指标——分类、回归、聚类、无监督

发表于 2019-06-04 | 分类于 机器学习 ， 评价指标

分类算法的评价指标：Accurary（准确率）、Precision（正确率、查准率）、Recall（召回率、查全率）、F-score、灵敏度、特效度、G-Mean、宏平均、微平均、ROC-AUC曲线、PR曲线、KS曲线（洛仑兹曲线）、Lift（提升图）、Gini系数、Kappa系数、PSI
回归算法评价指标：MAE（绝对误差）、MSE（均方误差）、RMSE（均方根误差）、$R_{2}$
聚类算法评价指标：兰德系数、调整兰德系数、互信息、轮廓洗漱、Calinski-Harabaz Index
无监督学习：密度或距离；统计学手段；转化成监督学习
